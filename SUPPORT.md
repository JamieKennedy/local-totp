# Support

Local TOTP is maintained on a best-effort basis with no guaranteed response or resolution time.

- Read the [documentation](https://jamiekennedy.github.io/local-totp/) and search existing issues before asking for help.
- Use GitHub Issues for reproducible bugs and focused feature requests.
- Use only synthetic examples. Never post seeds, current codes, passwords, recovery keys, API keys, backups, databases, or production account details.
- Report security vulnerabilities privately as described in [SECURITY.md](SECURITY.md).

Only the latest `1.x` release receives routine fixes. Older exact releases may be unsupported once a newer patch is available.
