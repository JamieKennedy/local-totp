# Governance

Local TOTP is currently a maintainer-led project. Jamie Kennedy is the sole maintainer, release manager, and final decision maker.

Contributors are welcome to propose changes through issues and pull requests. The maintainer evaluates proposals for fit with the project's localhost-only, synthetic-credential scope; security model; compatibility policy; and available maintenance capacity. No contribution or response time is guaranteed.

Material changes to cryptography, storage, authentication, the public interface, or major runtime dependencies require an architecture decision record. Releases follow semantic versioning and the process in [docs/RELEASING.md](docs/RELEASING.md).
